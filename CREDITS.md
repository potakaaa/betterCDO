## This Template

### What Changed

This template version includes the following modifications from the original:

1. **Framework Migration**: Converted from vanilla HTML/CSS/JS to Next.js 14 with React and TypeScript
2. **Templating System**: Added configuration-based system for any Philippine LGU
3. **LGU Type Support**: Added support for both municipalities and provinces
4. **Multi-language**: Added i18n support with template variable interpolation
5. **Setup Wizard**: Added interactive CLI wizard for easy configuration
6. **Removed Specific Data**: Replaced Solano-specific data with placeholder configurations

## Data Sources

The template is designed to work with data from official Philippine government sources:

- **PSA** (Philippine Statistics Authority) - Demographics and census data
- **BLGF** (Bureau of Local Government Finance) - Budget and financial data
- **DPWH** (Department of Public Works and Highways) - Infrastructure projects
- **DTI CMCI** (Competitiveness Index) - Municipal rankings

## How to Attribute

When using this template, please maintain attribution to:

1. **In your README**: Include a note that your site is "Based on [BetterGov LGU Template](https://github.com/BetterGov/betterlgu-template), forked from [BetterSolano](https://github.com/BetterSolano/bettersolano)"

2. **In your footer** (optional but appreciated): "Powered by BetterGov Template" or similar

3. **Keep the LICENSE file**: Maintain the original MIT license with attribution
